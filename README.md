# Catan Map Generator

A web app that generates a random Catan board layout — terrain, number tokens, and ports — following the official setup rules, so you can assemble the physical board from the result. It only generates the map; it does not implement gameplay.

## Tech stack

- Next.js (App Router, static export)
- TypeScript
- React
- Tailwind CSS
- Plain SVG for map rendering
- react-i18next (RU / EN / DE)
- Vitest
